##### World screen

![Imgur](https://i.imgur.com/Z7gcmHz.png)

Er is er voor gekozen om het World Screen ook gebruik te laten maken van de minigame engine aangezien daar components in staan die makkelijk hergebruikt konden worden. Hierbij kan er gedacht worden aan de buttons met zijn click methode, de background, de WPF controls en het gebruik van het entity components system.
